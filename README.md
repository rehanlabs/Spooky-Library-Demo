A **first-person 3D simulation prototype** where the player assumes the role of a librarian managing daily tasks in an eerie library.  
The project combines **routine simulation** with **atmospheric exploration** and light mystery elements.

  - Raycast-based object interaction with visual/audio feedback.  
  - Modular library environment with dynamic lighting and props.  
  - Subtle cues to create an **eerie atmosphere** and encourage exploration.  
- **Secrets & Exploration:** hidden clues and events integrated into library layout.  
- **UI:** minimal HUD showing tasks, objectives, and notifications.  
- **Persistence:** JSON-based save/load system for tasks and exploration progress.
